# Superstore Assistant using Flask, Ollama and ChromaDB

A beginner-friendly GenAI-powered retail analytics assistant built using:

* Flask
* Ollama
* ChromaDB
* Sentence Transformers
* RAG (Retrieval Augmented Generation)

## Features

* Natural language business questions
* Semantic search using embeddings
* Vector retrieval using ChromaDB
* LLM-generated responses using Ollama
* Local Flask deployment

## Example Questions

* Which region has the highest sales?
* What is the total sales?
* Which category performed best?

## Tech Stack

* Python
* Flask
* Ollama
* ChromaDB
* Sentence Transformers
* Pandas

## Run Locally

Install requirements:

```bash
pip install -r requirements.txt
```

Run Flask app:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
